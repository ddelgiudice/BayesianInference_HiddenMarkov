Calibr_SIPetc.r is the main script to perform Joint Inference of Input, Hydrological Model, and Output Error Parameters
with SIP. Other likelihood functions can also be chosen.

Refer to: 
Del Giudice, D., C. Albert, J. Rieckermann, and P. Reichert (2016), Describing the catchment-averaged precipitation as a 
stochastic process improves parameter and input estimation, Water Resour. Res., 52, 3162–3186, doi:10.1002/2015WR017871.
